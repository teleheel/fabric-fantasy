## Configuration
### SIHYWTCAMC and SIHYWTCAMD
This pack contains two mods that drastically changes many of vanilla's mob
mechanics as well as the vanilla combat. These mods are - [So I heard you were talking crap about Minecraft's difficulty?](https://www.curseforge.com/minecraft/mc-mods/sihywtcamd) and [So I heard you were talking crap about Minecraft's combat?](https://www.curseforge.com/minecraft/mc-mods/sihywtcamc), respectively.
All of the changes can be disabled, and comprehensive lists of the changes are on
the mod pages.

### Carpet
Check the github of [Carpet](https://github.com/gnembon/fabric-carpet) and
[Carpet Fixes](https://github.com/gnembon/carpet-extra) for a list of all settings.
To configure the mod, type `/carpet setDefault <setting> <value>` in the server
command line. You can also use the configuration included in this modpack by copying
the content of config/carpet/default_carpet.conf to world/carpet.conf. If the file
world/carpet.conf doesn't exist, create it.
