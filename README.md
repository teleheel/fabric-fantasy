## Configuration
### Cull Leaves
Cull leaves makes trees look ugly, and should only be enabled (through the
config) if you are experiencing perfomance issues or are using the Fast leaves
option in video settings.

### Fabrication
This mod includes a lot of different bugfixes, tweaks and new mechanics, all of
which can be configured. Just open the mod's config menu while not logged in to
a server to configure the client settings.
